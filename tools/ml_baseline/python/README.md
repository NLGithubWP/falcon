## Unit tests for Falcon's Cutom Implementation

to run unit-tests:

```sh
python3 -m unittest discover -s ./logreg_from_scratch/falcon/ -p "*_test.py"
```


or use `pytest`

```sh
# install pytest
pip3 install -U pytest
# check pytest version
pytest --version

# invoke by just
pytest
```
